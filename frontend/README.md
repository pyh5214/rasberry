# 프론트엔드

## 빠른 시작

```bash
# 1. 의존성 설치
npm install

# 2. 개발 서버 실행
npm start
```

애플리케이션은 `http://localhost:3000`에서 실행됩니다.

## 스크립트

- `npm start`: 개발 서버 실행
- `npm run build`: 프로덕션 빌드
- `npm test`: 테스트 실행
- `npm run electron-dev`: Electron 앱 개발 모드
- `npm run electron-pack`: Electron 앱 패키징

## 주의사항

백엔드 서버가 `http://localhost:5000`에서 실행 중이어야 합니다.
